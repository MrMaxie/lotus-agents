# Example 03 - Repository With Custom Paths

This example shows a repository that keeps durable docs and local operational
memory under custom directory names.

```text
repo/
  AGENTS.md
  README.md
  OVERVIEW.md
  knowledge/
    specs/
    meetings/
  workbench/
    AGENTS.local.md
    context.md
    issues/
    issue-notes/
    questions/
    runs/
  lotus.config.yaml
```

See the example `lotus.config.yaml` in this directory.
