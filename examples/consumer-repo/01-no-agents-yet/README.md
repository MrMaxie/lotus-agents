# Example 01 - No `AGENTS.md` Yet

Use this when the target repo does not have an `AGENTS.md`.

Copy `AGENTS_TO_COPY.md` and rename it to `AGENTS.md`.

```text
repo/
  AGENTS.md
  .local/
    context.md
    templates/
    issues/
    issues-notes/
```

Notes:

- `.local/` is the main private workspace
- create only the subdirectories you need
- add `docs/` later if the repo wants committed specs or meetings
