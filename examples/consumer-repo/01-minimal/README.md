# Example 01 - Minimal Repository

This example shows the smallest useful Lotus Agents adoption.

```text
repo/
  AGENTS.md
  README.md
  OVERVIEW.md
  INTEGRATION.md
  ARTIFACT_MATRIX.md
  CONTRACT_CHECKLIST.md
  skills/
  templates/
```

Characteristics:

- no `docs/`
- no `.local/`
- no `lotus.config.yaml`
- the agent works from the contract, codebase, diff, and current human input
