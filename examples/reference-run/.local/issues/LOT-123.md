# Issue - LOT-123

## Goal
Reduce noisy search results when multiple repositories match a loose query.

## Background
Users reported that the current search flow guesses the repository too early.

## Constraints
- Do not guess when multiple repository identifiers are plausible.
- Keep committed specs as the durable source of truth.

## Acceptance Criteria
- Exact repository filters are used when available.
- Ambiguous repository identifiers trigger clarification instead of guessing.

## Open Questions
- Whether a fallback suggestion list should be shown before confirmation.
