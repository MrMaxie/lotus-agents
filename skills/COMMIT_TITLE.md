# SKILL: COMMIT_TITLE

Optional skill. It can live in the repo or be installed globally on the
machine.

## INPUT
- diff
- issue
- specs

## OUTPUT
- 1-3 commit title proposals

## RULES

- follow repo patterns
- include intent, not implementation
- concise

## FORMAT

<type>: <what changed>

example:
feat: add collapsible chat with dual state
